Reservation Details

Help and Support

The Reservation Details refer to the API's documentation.
In order to make an API request to the endpoint, the user needs to fill the values for, "Arrival Airport", and "Departure Date". These values create the "body" that is later sent to the endpoint.

Based on the date selected, the corresponding data will be fetched from the endpoint. The user is responsible for what data they want to fetch from the database.

Note: Upon submit, wait a couple of seconds until the download starts. In case you cannot see the download, check the browser's settings, or try clearing your browser's cache.
