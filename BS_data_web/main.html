Help and Support

The Reservation Details, reffer to the Blue Style's API documentation.
In order to make an API request to the endpoint, the user needs to fill the values for "Departure Airport", "Arrival Airport", and "Departure Date". Those values create the "body" that is later sent to the endpoint.

Based on the date selected, the according data will be fetched form the endpoint. The user is responsible, for what data they want to fetch from the database.

Note: Uppon submit, wait a couple of seconds until the downlaod starts. In case you cannot see the downlaod, check the browser's settings, or try clearing your browser's chache .
